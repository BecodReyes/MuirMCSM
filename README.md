MuirMCSM
--------
> 这是一个基于 Shell 的 Minecraft 服务端控制工具

## Usage
- 配置.config
- 运行mc.sh

## Project
- Docker
- 群组服支持
- 全模块化 (是的没错现在半模块化只是方便区分功能)
- 优化亿下代码

## Used open source projects
- [ttyd](https://github.com/tsl0922/ttyd)
